## 개요
마크다운(Markdown)은 기본 문법만 알고있다면 일반 텍스트 편집기에서도 손쉽게 작성 가능한 마크업 언어이다.  
HTML과 리치텍스트 (RTF) 등 서식 문서로 쉽게 변환되므로, 응용소프트웨어와 함께 배포되는 README파일 이나 온라인 게시물 등에 많이 사용된다.
